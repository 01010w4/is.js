is.js
=====

Simple data checks

** Type checks: **
- array
- object
- function
- string
- number
- boolean
- regexp
- nan
- null
- undefined
- date
- sameType    // will work with NaN too

** Value checks: **
- existy
- truthy
- falsy

** Regexp checks: **
- email
- url
- date
- creditCard

** Environment checks: **
- browser
  - chrome
  - firefox
  - ie
  - opera
  - safari
- device
  - mobile
  - tablet
  - desktop
- os
  - linux
  - mac
  - windows
